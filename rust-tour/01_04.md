## 変数の変更

Rustは変数の変更についてとても配慮しています。値は 2
種類に分類されます。

-   **可変** - コンパイラは変数の書き込みと読み込みを許可します。
-   **不変** - コンパイラは変数の読み込みだけを許可します。

可変値は **mut** キーワードで表します。

この概念については後で詳しく説明しますが、今のところはこのキーワードに注意してください。

```
fn main() {
    let mut x = 42;
    println!("{}", x);
    x = 13;
    println!("{}", x);
}
```
