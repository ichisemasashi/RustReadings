# [標準ライブラリの型](#標準ライブラリの型){.header}

`std`ライブラリは、基本データ型を劇的に拡張するカスタム型を数多く提供します。例えば以下です。

-   拡張可能な文字列である`String`。例えば: `"hello world"`
-   オプション型: `Option<i32>`
-   エラーハンドリング用の`Result<i32, i32>`
-   ヒープ上資源のポインタ`Box<i32>`
